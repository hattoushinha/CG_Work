CG Works　「自作イラスト」
==========================

このリポジトリは自作のイラストを管理する目的で作られました。

ソフト独自形式とPNG画像の両方でログをとっています。

年単位でフォルダ分け、名前で日付分けをしてあります。


画像のログ画面の下部にある4つのリンクで画像の変更箇所がわかりやすく確認できます。
